## Step 1 - Choose a website to deploy your web app (Glitch, Replit)
You can either use a paid website or the following free ones: Glitch, Replit <br> For this tutorial, I will be using **Glitch**. </br>

> 1) Sign up and create a new project

Navigate to `https://glitch.com` in your browser:
![image](https://github.com/syflairenicole/rbxcaptcha/assets/105584041/e46fec81-6729-4f5b-9d7f-cff876fd5de9)

To sign up, click **'Sign Up'** bottom at the top of the screen:
![image](https://github.com/syflairenicole/rbxcaptcha/assets/105584041/aefa1bed-7964-4594-bf8d-f116cc7e3d73)

Now that you've got that sorted out, you wanna go ahead and create a new project. <br>Find the **'New Project'** button at the top of the screen:</br>
